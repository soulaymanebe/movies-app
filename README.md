# 🎬 Movie Streamer

A basic Python Flask app to stream movies.

## Setup

### 1. Get OMDb API Key
Visit [OMDb API](https://www.omdbapi.com/apikey.aspx) an get your free API key.
Update 'Add_Your_OMDb_Token' in app.py with the token you received.

### 2. Install dependencies

Clone this repo and install dependencies:

```bash
cd movie-streamer
pip install -r requirements.txt
```

### 3. Start the app

```bash
python app.py
```

### 4. Access the app
http://127.0.0.1:5000
