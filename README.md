# 🎬 Movie Streamer

A basic Python Flask app to stream movies using Flask.

## Setup

### 1. Get OMDb API Key
Visit [OMDb API](https://www.omdbapi.com/apikey.aspx) to get your free API key.
Update 'Add_Your_OMDb_Token' with the token you received.

### 2. Install Python3 and Dependencies

Clone this repo and install dependencies:

```bash
git clone https://github.com/soulaymanebe/movies-app
cd movie-streamer
pip install -r requirements.txt
```

```bash
# Run the app
python app.py
```

Access the app at http://127.0.0.1:5000.
